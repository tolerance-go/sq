# 归并排序

## 分析

| 时间复杂度（平均/最好/最坏）               | 空间复杂度 | 排序方式 | 稳定性 |
| ------------------------------------------ | ---------- | -------- | ------ |
| O(nlogN) / O(nlogN) / O(nlogN) | O(n)       | Out-place | 稳定 |

## 思路

定义一个处理左右数组的函数 merge，主函数中截断数组得到2个数组，进行合并并返回，结束条件是 len < 2
