# 插入排序

## 分析

| 时间复杂度（平均/最好/最坏）               | 空间复杂度 | 排序方式 | 稳定性 |
| ------------------------------------------ | ---------- | -------- | ------ |
| O(n<sup>2</sup>) / O(n) / O(n<sup>2</sup>) | O(1)       | In-place | 稳定 |

## 思路

不断消费未排序的数组元素，在已经排序的数组找到合适的一个位置插入
